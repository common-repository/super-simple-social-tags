== Description ==

Super Simple Social Tags is a super lightweight plugin to add social media meta tags for Facebook and Twitter. Adds tags (in document <head>) to display the Wordpress featured image in your social media posts. No settings page necessary. Just activate and go.

Perfect for sites that use the Genesis framework and would rather not install bloated SEO plugins with useless features.

== Installation ==

1. Upload the entire super-simple-social-tags folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. Whenever your post is shared on Facebook or Twitter, the featured image for the post is displayed.
